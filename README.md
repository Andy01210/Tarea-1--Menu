Enunciado
Desarrolle una aplicación de consola en Java que contenga un menú interactivo con múltiples opciones. Cada opción del menú debe ejecutar uno de los ejercicios asignados durante la práctica de los capítulos 1 al 5 del libro Java 2 Manual de Programación.

La aplicación debe permitir al usuario navegar entre las diferentes funcionalidades hasta que seleccione la opción de salida.

Opciones del menú
Calculadora básica
Número par o impar
Tabla de multiplicar
Contador de vocales
Promedio de notas
Clase Estudiante
Cuenta bancaria
Herencia de vehículos
Matriz 3x3
Sistema básico de inventario
Salir de la aplicación


Requisitos obligatorios

Crear un menú utilizando switch.
Cada ejercicio debe estar separado en métodos diferentes.
Utilizar buenas prácticas de programación.
Mantener una correcta indentación del código.
Comentar las partes importantes del programa.
Utilizar Programación Orientada a Objetos donde aplique.
La aplicación debe ejecutarse continuamente hasta seleccionar la opción 11.
El proyecto debe compilar y ejecutarse correctamente en Java.
